Installation and configuring

To get the PlebTap up and running, you need to perform the following steps: Flash the firmware, Configure LNbits and Configure the PlebTab.

Flash the Firmware

To install a fresh firmware, use the following steps:

  1. Use a compatible browser such as Chrome or Edge.
  2. Unplug the device (the Lilygo T-Display S3).
  3. Reconnect the device while holding the 'Boot' button (left from the USB connector)
  4. Release the 'Boot' button.
  5. Select the firmware and press the 'Flash' button to start the flashing process.
  6. After flashing is completed. Reset or power cycle the device.

Configure LNbits

LNbits takes care of the payment, and notifying the PlebTap after a succesfull payment is completed. You can use your own instance of LNbits. As an alternative, it is also possible to use the public testing instance of LNbits: https://legend.lnbits.com/. To configure LNbits, take the following steps:

  1. Log in to your wallet in LNbits and verify that the LNURLdevice extension is enabled. If not, click 'Manage extensions' and enable the 'LNURLDevice' extension.
  2. Open the LNURLDevice extension and create a new device by clicking on the 'NEW LNURLDEVICE INSTANCE'.
  3. Fill in the required information in the pop-up window. Select 'Switch', enter an amount and enter some data for millisecs and GPIO (e.g. 1 and 1).
  4. Click 'CREATE LNURLDEVICE'.
  5. The LNURL string is needed for PlebTap confguration. To copy this string, click the 'eye' icon and in the pop-up window copy the LNURL string.
  6. The LNURL Device String (or WebSocket link) is also needed for PlebTap configuration. To copy this string, click the 'LNURLDevice settings' icon. Copy the string from the pop-up window.

PlebTap Configuation

To configure the device, use the following steps:

  1. Use a compatible browser such as Chrome or Edge.
  2. Connect the device to the USB port.
  3. Click the 'Show Config' button to open the configuration interface.
  4. Click 'Select Template' and select 'PlebTap Config' to load the configuration template.
  5. Click the 'Connect' button and select the serial port of the device.
  6. (Optional) Load the current configuration by clicking 'Download from Device'.
  7. Modify the configuration and enter the LNURL and WebSocket link from the LNbits configuration.
  8. Click 'Save to Device' to upload the configuration to the device. When upload is completed, the configuration is directly loaded into the device.
